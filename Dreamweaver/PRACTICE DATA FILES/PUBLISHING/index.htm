Publishing a Web Site

Publishng a Web site is a process that involves copying the Web site folder contents to a destination. Before the files can be copied, the site needs to be prepared for publishing and a Web host needs to be located.

Once a site is published, it can be viewed in a Web browser. Publishing on the World Wide Web means the information in the site has the potential to be viewed by millions. To keep users coming back to the site, content must be updated on a regular basis and feedback monitored to know what your audience thinks about the Web site.
