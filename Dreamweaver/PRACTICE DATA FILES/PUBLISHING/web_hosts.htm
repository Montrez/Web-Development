Web Hosts

Web servers are often managed by web hosting companies, which provide space on their server for a fee.
Web host services include registering a domain name for a web site. Domain names are used to identify a particular web page and are made up of a seqence of parts, separated by a period that may stand for the server, organization, and organization type. For example, www.lpdatafiles.com.

A domain name should be something users associate with the web site content, such as the business name, and it should be something easy to remember. Registering a domain name provides the exclusive right to use that name. A periodic renewal fee is required to keep the domain name.
